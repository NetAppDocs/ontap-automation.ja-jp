= REST API の使用を開始する
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP REST API などの自動化テクノロジをすぐに使用を開始できます。基本的な要求と応答のフローを準備して理解したら、 cURL を使用して問題 呼び出しを実行できます。この準備では、稼働中のシステムでより複雑なワークフローを使用する前に、必要な概要と展望を説明します。
