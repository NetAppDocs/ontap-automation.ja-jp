= 概要
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
ONTAPI APIを使用する場合は、ONTAP REST APIへの移行を開始することを推奨します。移行作業に役立つリソースやツールが多数用意されています。


NOTE: ネットアップはまず、ONTAP 9.6でREST APIを導入しました。以来、APIは拡張と成熟を続けています。ONTAPIに代わる戦略的な選択肢として、ONTAP 環境の管理を自動化できるようになりました。
