= REST API の使用を開始する
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
ONTAP REST API はすぐに使用を開始できます。基本的な要求と応答のフローを準備して理解したら、 curl を使用して簡単な呼び出しを問題できます。これにより、稼働中のシステムでより複雑なワークフローを使用する前に、必要な概要と視点を確認できます。
